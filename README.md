## **Installing this config**

-  **For Window**

  >  cd C:\Users\YourName\AppData\Local\

  >  git clone https://github.com/Top-Slayer/my-neovim-config.git

  >  rename my-neovim-config nvim

- **For Linux**

  >  cd ~/.config/
  
  >  git clone https://github.com/Top-Slayer/my-neovim-config.git
  
  >  mv my-neovim-config nvim
